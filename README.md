# Simple Interest Calculator

## Description
This project provides a simple interest calculator that computes the simple interest 
based on user input. It is implemented as a Bash script.

## Simple Interest Formula
Simple Interest (SI) = (Principal × Rate × Time) / 100

Where:
- **Principal (P)**: The initial amount of money
- **Rate (R)**: The annual rate of interest (in percentage)
- **Time (T)**: The time period (in years)

## Example
- Principal = 1000
- Rate = 5%
- Time = 3 years
- Simple Interest = (1000 × 5 × 3) / 100 = **150**

## Usage Instructions
1. Clone the repository:
```bash
   git clone https://github.com/your-username/your-repo.git
```
2. Navigate to the project directory:
```bash
   cd your-repo
```
3. Run the script:
```bash
   bash simple-interest.sh
```
4. Enter the required inputs when prompted:
   - Principal amount
   - Rate of interest (%)
   - Time period (years)
5. The script will display the calculated Simple Interest.
